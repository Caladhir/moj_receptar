# MojReceptar

Recipe management and meal planning app built with C++ Builder.

## Features

- **Database** - MS Access with master-detail relations, BLOB support
- **File formats** - XML (recipes), JSON (ratings), binary (nutrition)
- **Networking** - TCP/UDP client-server communication
- **Multithreading** - Parallel tasks with synchronization (critical sections, events)
- **Reports** - PDF generation with FastReport
- **Localization** - Croatian/English

## Tech Stack

C++ Builder | VCL | ADO | Indy | FastReport
